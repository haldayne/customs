Get the HTML variable name for this upload.

This name approximates the originally given name, like:

But in some cases the exact name cannot be returned. First, because PHP mangles HTML names containing '.' or ' ', these will map: * foo.bar returned as "foo_bar" * foo bar returned as "foo_bar" * foo_bar returned as "foo_bar" * foo[bar.baz] returned as "foo[bar.baz]"

Second, if the auto-append syntax was given: * foo[] returned as "foo[0]", etc.
